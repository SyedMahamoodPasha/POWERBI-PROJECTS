# VEBY Supermarket Sales Dashboard

## Project Overview

The **VEBY Supermarket Sales Dashboard** is an interactive **Power BI Business Intelligence project** designed to analyze supermarket sales performance and generate actionable business insights. The dashboard enables users to monitor key performance indicators, compare category performance, analyze item visibility, and identify sales trends using dynamic filters and visual analytics.

## Key Features

* **KPI Cards:** Total Sales, Average Sales, and Number of Items.
* **Sales Trend Analysis:** Line chart for tracking sales performance over time.
* **Category Performance:** Bar charts for comparing sales across product categories and outlets.
* **Sales Distribution:** Donut charts showing contribution by category/outlet.
* **Item Visibility Analysis:** Pivot table to identify products with high and low visibility.
* **Funnel Analysis:** Visual representation of performance stages.
* **Interactive Slicers:** Dynamic filtering for customized analysis.

## Tools & Technologies

* **Power BI Desktop**
* **DAX (Data Analysis Expressions)**
* **Power Query**
* **Excel / Retail Sales Dataset**

## Business Insights

* Identified top-performing product categories.
* Compared sales contribution across different outlets.
* Analyzed item visibility and its impact on sales performance.
* Tracked sales trends to support inventory and business planning.
